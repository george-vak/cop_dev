##############
Mood Server
##############
**Постановка задачи**

Разработать серверную часть Mood, обеспечивающую:

- **Асинхронная обработка клиентских подключений**
  - Неблокирующий I/O на основе asyncio

- **Обработка пользовательских команд**:

  *Движение*:
    - ``up/down/left/right`` - перемещение на 1 позицию

  *Боевые действия*:
    - ``attack <monster>`` - базовая атака монстра по имени
    - ``attack <monster> with <оружие из арсенала>`` - атака с оружием

  *Добавление монстра*:
    - ``addmon <name> hello <hello word> hp <hp> coords <x> <y>`` - добавление монстра на поле 

- **Логирование операций**
